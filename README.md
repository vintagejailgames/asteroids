## Controls
 * `cursor amunt` accelerar
 * `cursor avall` frenar
 * `cursor dreta` rotar en el sentit de les agulles del rellotge
 * `cursor esquerra`rotar en l'altre sentit
 * `espai` disparar
 * `esc` eixir
